    # 🚀 MKFiloServis Programı Çalıştırma Talimatı

---

## 🔴 1. KURULUM

1. Size gönderilen **MKFiloServisKurulum.exe** dosyasını çalıştırın
2. Kurulum tamamlanana kadar **İleri → İleri → Kur** adımlarını takip edin
3. Kurulum sonrası program otomatik açılacaktır

---

## 🔴 2. PROGRAMI BAŞLATMA

Eğer otomatik açılmazsa:

👉 Masaüstünden veya Başlat menüsünden:

```
MKFiloServis
```

ikonuna çift tıklayın

---

## 🔴 3. İLK AÇILIŞ (LİSANS EKRANI)

Program ilk açıldığında aşağıdaki ekran gelir:

### Yapmanız gerekenler:

1. **Makine Kodunu Kopyalayın**
2. Bu kodu yetkili kişiye gönderin
3. Size verilen **Lisans Anahtarını** alın
4. Anahtarı ilgili alana yapıştırın
5. **"Lisansı Aktif Et ve Devam Et"** butonuna basın

---

## 🔴 4. LİSANS AKTİF EDİLDİKTEN SONRA

✔ Sistem otomatik açılır
✔ Giriş ekranı gelir
✔ Kullanıcı adı ve şifre ile giriş yapılır

---

## 🔴 5. PROGRAM NASIL ÇALIŞIR?

```text
✔ Program arka planda çalışır
✔ Tarayıcı otomatik açılır
✔ Sistem internet olmadan da çalışabilir
```

---

## 🔴 6. PROGRAMI TEKRAR AÇMA

Programı tekrar açmak için:

```text
Masaüstü → MKFiloServis → Çift tık
```

---

## 🔴 7. OLASI SORUNLAR

### ❌ Tarayıcı açılmazsa:

Tarayıcıya yazın:

```
http://localhost:5190
```

---

### ❌ Program açılmıyorsa:

```text
Bilgisayarı yeniden başlatın ve tekrar deneyin
```

---

### ❌ Lisans hatası:

```text
Doğru lisans anahtarını kullandığınızdan emin olun
```

---

## 🔴 8. DESTEK

Herhangi bir sorun yaşarsanız:

```text
Makine Kodunu göndererek destek isteyin
```
